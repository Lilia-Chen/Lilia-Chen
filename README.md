## Open Source

### AIRI
Core contributor to **[AIRI](https://github.com/moeru-ai/airi)** — an extensible platform for building real-time AI agents with avatar interfaces.

Work includes:
- VRM avatar scene system
- gaze tracking & camera control
- lip-sync and animation pipeline
- MediaPipe motion capture integration
- rendering and shader system improvements

### plast-mem
Co-creator of **[plast-mem](https://github.com/moeru-ai/plast-mem)** — a biologically-inspired memory system for AI agents.

Features:
- episodic + semantic memory architecture
- episodic segmentation of interaction logs
- hybrid retrieval (BM25 + vector + RRF)
- memory review loop with FSRS retrievability
